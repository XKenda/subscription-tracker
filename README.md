# Subscription Tracker

A lightweight, efficient JavaScript application for managing and tracking your subscriptions in one centralized place.

## Overview

Subscription Tracker helps you keep track of all your subscriptions—whether they're software services, streaming platforms, SaaS tools, or memberships. Never miss a renewal date and easily visualize your spending patterns with an intuitive interface.

## Features

- 📋 **Centralized Management** - Keep all your subscriptions in one place
- 💰 **Cost Tracking** - Monitor your subscription spending over time
- 🔔 **Renewal Reminders** - Get notified before your subscriptions renew
- 📊 **Analytics Dashboard** - Visualize spending patterns and trends
- 🏷️ **Categorization** - Organize subscriptions by type or category
- 🔍 **Search & Filter** - Quickly find subscriptions

- 🔐 **Secure Storage** - Your data stays private

## Getting Started

### Prerequisites

- Node.js 14.x or higher

### Quick Start

```bash
# Clone the repository
git clone https://github.com/XKenda/subscription-tracker.git

# Navigate to the project directory
cd subscription-tracker

# Install dependencies
npm install

# Start the development server
npm start
